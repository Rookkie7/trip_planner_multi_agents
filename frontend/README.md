# 智能旅行规划助手 - 前端

基于 React + TypeScript + Vite 构建的智能旅行规划助手前端应用。

## 功能特性

- **双重输入模式**
  - 自然语言输入：用户可以用自然语言描述旅行需求
  - 表单输入：结构化表单输入，精确控制各项参数

- **智能行程规划**
  - 景点推荐（含图片、地址、门票价格）
  - 餐饮建议（早中晚餐推荐）
  - 酒店推荐
  - 天气信息
  - 预算管理

- **美观的用户界面**
  - 响应式设计，支持移动端和桌面端
  - 现代化的 UI 设计
  - 流畅的动画效果

## 快速开始

### 安装依赖
\`\`\`bash
npm install
\`\`\`

### 启动开发服务器
\`\`\`bash
npm run dev
\`\`\`

应用将在 http://localhost:5173 启动。

### 构建生产版本
\`\`\`bash
npm run build
\`\`\`
